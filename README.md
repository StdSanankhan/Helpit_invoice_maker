 Helpit – Smart Invoice Maker

Helpit is a modern web-based invoice maker designed for freelancers and small businesses.

 Features

 Create professional invoices
 Client management system
 Automatic total and tax calculation
 Export invoices as PDF
 Send invoices via email
 Invoice templates and status tracking
 Dashboard analytics

Tech Stack

Frontend: React.js
Backend: Python
Database: JSON (file-based NoSQL)

 Purpose

This project is developed as a university project to demonstrate a SaaS-style invoice management system.
